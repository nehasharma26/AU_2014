1) index.jsp is the first page which has links to add institute, add courses and view list of courses.
2) addInstitute allows the user to add new institute using a form, which calls a servlet.The servlet uses hibernate to add institute into database.
3) addcourse allows the user to add new course using a form, which calls a servlet.The servlet uses hibernate to add institute into database.
4) institutes.jsp displays list of all institute, and provides links to view details of institute and courses.
5) the page showing details of institute also allows the user to delete and update info.
